**Plugin Implementation Details:**

Configuration Options (from config/options.php):
- getkirby-uniform.mosparo.host: Mosparo instance URL
- getkirby-uniform.mosparo.uuid: Project UUID
- getkirby-uniform.mosparo.publicKey: Public key (frontend)
- getkirby-uniform.mosparo.privateKey: Private key (server-side, secret)
- getkirby-uniform.mosparo.ignoredFields: Fields to exclude from verification
- getkirby-uniform.mosparo.cssUrl: Custom CSS URL
- getkirby-uniform.mosparo.debug: Debug mode

Template Helpers (from src/helpers.php):
- mosparo_field(array $options): Renders widget field
- mosparo_script(array $options): Renders script tag

Form Trait Methods (from src/Form/FormExtensions.php):
- $form->mosparoField(): Render field via trait
- $form->mosparoScript(): Render script via trait

Widget Renderer (from src/Widget/WidgetRenderer.php):
- Static methods for snippet usage
- Data attributes support

Available Snippets:
- snippets/mosparo-field.php: Widget container with data attributes
- snippets/mosparo-script.php: JS/CSS loading with async/defer options

Error Messages (from i18n/en.php):
- mosparo.error.not_configured
- mosparo.error.tokens_missing

- mosparo.error.api_error
