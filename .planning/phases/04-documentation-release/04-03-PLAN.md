   - Overview of differences
   - Step 1: Remove hCaptcha script tags
   - Step 2: Replace field names (h-captcha-response → _mosparo_submitToken + _mosparo_validationToken)
   - Step 3: Update Guard (HCaptchaGuard → MosparoGuard)
   - Step 4: Configure Mosparo credentials
   - Code before/after comparison

4. **Migrating from Cloudflare Turnstile**:
   - Overview of differences
   - Step 1: Remove Turnstile script tags
   - Step 2: Replace field names (cf-turnstile-response → _mosparo_submitToken + _mosparo_validationToken)

  <name>Create TROUBLESHOOTING.md for common errors</name>
  <files>docs/TROUBLESHOOTING.md</files>
  <action>
Create docs/TROUBLESHOOTING.md addressing common errors and their solutions.

Structure:

1. **Introduction**: How to use this guide, enabling debug mode

2. **Widget Not Loading**:
   - Symptom: No widget appears on the page
   - Causes:
     * Mosparo not configured (missing host/UUID/keys)
     * Script not included
     * JavaScript errors
   - Solutions for each cause
   - How to check: browser dev tools, network tab

3. **Form Submissions Failing**:
   - Symptom: Form always shows error after submission
   - Causes:
     * Missing tokens (widget not completed)
     * Invalid tokens (expired)
     * API errors (connection issues)
   - Solutions for each
   - Error message reference (link to i18n keys)

4. **Configuration Errors**:
   - "Mosparo is not configured" error
   - Wrong host URL
   - Invalid UUID or keys
   - How to verify credentials

5. **API Connection Issues**:
   - Symptom: "Unable to verify spam protection" error
   - Causes:
     * Network/firewall blocking
     * Wrong host URL
     * SSL certificate issues
   - Solutions and debugging steps

6. **Bypass Detection Errors**:
   - Symptom: "Form submission could not be verified"
   - Cause: Spam bot detected or field manipulation
   - Solution: Check for JavaScript tampering, verify widget initialization

7. **Debug Mode**:
   - How to enable: 'getkirby-uniform.mosparo.debug' => true
   - What logs to check
   - Understanding error messages

8. **Getting Help**:
   - How to report issues
   - Information to include (versions, config, error logs)
   - Links to Mosparo documentation

For each error:
- Clear symptom description
- Likely causes
- Step-by-step solutions
- Code examples where relevant

Reference error keys from @i18n/en.php and exception codes from @src/Exception/VerificationException.php
