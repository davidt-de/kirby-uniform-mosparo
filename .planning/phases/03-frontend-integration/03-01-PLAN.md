<task type="auto">
  <name>Create Form extensions trait</name>
  <files>src/Form/FormExtensions.php</files>
  <action>
    Create `src/Form/FormExtensions.php` as a PHP trait that can be used with Uniform Form:

    ```php
    namespace Uniform\Mosparo\Form;

    trait FormExtensions
    {
        /**
         * Render Mosparo widget field
         *
         * @param array $options Widget options (id, class, data-attributes)
         * @return string HTML for the widget
         */
        public function mosparoField(array $options = []): string
        {
            return mosparo_field($options);
        }

        /**
         * Render Mosparo script tag
         *
         * @param array $options Script options (async, defer, id)
         * @return string HTML script tag
         */
        public function mosparoScript(array $options = []): string
        {
            return mosparo_script($options);
        }
    }
    ```

    The trait should:
    - Use proper namespace Uniform\Mosparo\Form
    - Include PHPDoc with @param and @return
    - Delegate to the helper functions (don't duplicate logic)
    - Return string type (not echo)
  </action>
  <verify>
    <automated>php -l src/Form/FormExtensions.php</automated>
    <manual>Verify trait structure matches Kirby Uniform patterns</manual>
  </verify>
