  <action>
Create `ConfigFactory` class in `src/Config/ConfigFactory.php` that:

1. Has a static method `fromKirbyOptions(): Config` that:
   - Gets Kirby app instance via `kirby()` helper or `App::instance()`

     - uuid
     - publicKey

     - debug (bool, defaults to false)

2. Has a static method `create(array $options): Config` for testing that creates Config directly from array

3. Handles null/empty values gracefully (allows partial config for testing)
