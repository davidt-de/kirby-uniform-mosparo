2. **perform() method implementation:**
   ```php
   public function perform(): void
   {
       // Load configuration
       $config = ConfigFactory::fromKirbyOptions();
       
       // Check if configured
       if (!$config->isConfigured()) {
           throw new VerificationException(
               VerificationException::NOT_CONFIGURED,
               'Mosparo is not configured. Please set host, uuid, publicKey, and privateKey options.'
           );
       }
       
       // Extract Mosparo tokens from form data
       $submitToken = $this->data['_mosparo_submitToken'] ?? null;
       $validationToken = $this->data['_mosparo_validationToken'] ?? null;
       
       if (empty($submitToken) || empty($validationToken)) {
           $this->reject(
               'mosparo.error.tokens_missing',
               'mosparo'
           );
           return;
       }
       
       // Create Mosparo API client
       $client = new Client(
           $config->getHost(),
           $config->getPublicKey(),
           $config->getPrivateKey()
       );
       
       // Prepare form data (remove Mosparo tokens)
       $formData = $this->data;
       unset($formData['_mosparo_submitToken'], $formData['_mosparo_validationToken']);
       
       try {
           // Verify submission
           $result = $client->verifySubmission($formData, $submitToken, $validationToken);
       } catch (Exception $e) {
           // Log error (without private key details)
           error_log('Mosparo API error: ' . $e->getMessage());
           
           $this->reject(
               'mosparo.error.api_error',
               'mosparo'
           );
           return;
       }
       
       // Check if submission is valid
       if (!$result->isSubmittable()) {
           $this->reject(
               'mosparo.error.verification_failed',
               'mosparo'
           );
           return;
       }
       
       // Success - do nothing (Guard passed)
   }
   ```

3. Add private helper method `prepareFormData(array $data, Config $config): array` that:
   - Removes Mosparo tokens
   - Removes ignored fields (from config)
   - Returns cleaned data

4. Update docblock to reflect actual implementation

5. Keep strict typing: declare(strict_types=1)
